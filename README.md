# setup-macos
Setup files for MacOS


## Prepare Install
### Install Brew

https://brew.sh/

### Install Mas

https://github.com/mas-cli/mas

### Install DevTools

https://github.com/nvm-sh/nvm

## Install Applications

```
$ brew bundle install
$ brew bundle cleanup --force 
```

## Setting Dock/Finder
```
$ ./bootstrapMacOS.sh
$ ./bootstrapThirdPartyApp.sh
```
